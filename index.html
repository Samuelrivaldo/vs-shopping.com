🛒 VS Shopping

Plateforme d'E-commerce Sécurisée & Moderne

ℹ️ À Propos

VS Shopping est une application web d'e-commerce complète développée en PHP avec une architecture moderne et sécurisée.

Cette page est un aperçu du projet. L'application complète nécessite un serveur avec PHP pour fonctionner.

✅ Dernière Mise à Jour - Sécurité Améliorée

L'application inclut maintenant :

  • 🔐 Configuration centralisée et sécurisée (.env)
  • 🛡️ Protection CSRF sur tous les formulaires
  • 🔒 Prepared Statements pour éviter les injections SQL
  • 🔑 Hachage sécurisé des mots de passe (bcrypt)
  • 📁 Validation stricte des uploads de fichiers
  • ✨ Sanitization XSS complète

🌟 Fonctionnalités Principales

🛍️ Gestion Produits

Créer, modifier et supprimer des produits avec images et descriptions complètes.

👥 Systèmes d'Authentification

Inscription et connexion sécurisées avec hachage de mots de passe.

🔐 Architecture Sécurisée

Prepared statements, CSRF tokens, sanitization XSS et validation d'uploads.

📊 Tableau de Bord Admin

Interface administrative pour gérer les produits et les catégories.

📱 Responsive Design

Interface adaptée à tous les appareils (mobile, tablette, desktop).

🗄️ Base de Données

MySQL avec architecture optimisée pour la performance.

🚀 Déploiement

Pour déployer VS Shopping sur un serveur :

  1. Cloner le repository :
    git clone https://github.com/Samuelrivaldo/vs-shopping.com.git
  2. Configurer les variables d'environnement :

    Créer un fichier .env à la racine avec :

    DB_HOST=votre_host DB_NAME=vs-shoppingg DB_USER=votre_user DB_PASSWORD=votre_password APP_ENV=production APP_DEBUG=false
  3. Créer la base de données :

    Importer le schéma MySQL fourni

  4. Accéder à l'application :

    Le serveur doit pointer vers la racine du projet

📋 Prérequis Système

  • PHP 7.4+ (avec PDO MySQL activé)
  • MySQL 5.7+
  • Apache/Nginx avec support .htaccess
  • OpenSSL pour HTTPS
⚠️ Note Importante

GitHub Pages ne supporte que les fichiers statiques. Cette page HTML est un aperçu du projet. Pour utiliser l'application complète, vous devez la déployer sur un serveur avec PHP.
